# Football Team Cards - FC Barcelona Edition

A dynamic web application that displays player information and allows users to filter the squad by position. 

## 🚀 Features
- **Dynamic Squad Rendering:** Automatically generates player cards from a JavaScript object.
- **Real-time Filtering:** Uses a dropdown menu to filter players by position (Forward, Midfielder, Defender, Goalkeeper).
- **Captain Highlights:** Automatically identifies team captains within the UI.
- **Responsive Design:** Optimized for both desktop and mobile viewing.

## 🛠 Technologies Used
- **HTML5:** Semantic structure for the team dashboard.
- **CSS3:** Custom styling for the player cards and layout.
- **JavaScript (Vanilla):** Implements `filter()`, `switch` statements, and DOM updates without external libraries.
